Client README: run `npm install` then `npm run dev`.

Development quick start:
1. Install dependencies

```bash
cd client
npm install
```

2. Start Vite dev server

```bash
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173). Do not use static Live Server on the repo root — Vite serves ES modules and dev transforms.

If you see 404s for `/src/main.jsx` or `/favicon.ico`, ensure you're opening the Vite URL and not another server (e.g. :5500).
